## ğŸš€ Getting Started

### Prerequisites

- Flutter SDK (3.9.2 or higher)
- Chrome browser for web development

### Installation

1. Clone the repository:
```bash
git clone https://github.com/waghmarepb/sagar-portfolio.git
cd sagar-portfolio
```

2. Install dependencies:

3. Run on web:
```bash

```
